====================================================
CORE PROJECT CONCEPT
====================================================

The platform architecture works like this:

Frontend Website (HTML/CSS/JavaScript)
        ↓
Prompt sent to AI API
        ↓
Google Colab GPU / Cloud GPU generates image
        ↓
Stable Diffusion model processes image
        ↓
Generated image returned
        ↓
Website displays downloadable wallpaper

The heavy AI rendering MUST happen in cloud GPUs.

This project is NOT a demo.
This MUST be developed as a REAL scalable SaaS-style AI platform.
